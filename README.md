## Project Details
#
Simple tour package booking has been worked on in this repository.
#
### `Technology Use`
#
1. React JS
2. React Hooks
3. Event Handlers
4. Arrow Functions
5. Fetch API
6. React Router
7. React Router Dom
8. React Private Route
9. React Lists and Keys
10. React Alert
11. Firebase
12. Firebase Authentication
13. Google Authentication
14. Node JS
15. Express JS
16. Express Middleware
17. Express CORS
18. Nodemon
19. MongoDB
20. MongoDB Atlas
21. Font Awesome
22. Google Fonts
23. HTML5
24. CSS3
25. Bootstrap5

#
### `Project Deploy`
#
1. Firebase Hosting (Frontend Part)
2. Vercel (Backend Part)
